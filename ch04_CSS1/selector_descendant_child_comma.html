[selector(선택자)]
div > p : child(자식 선택자)에서만 css적용
div,p : group(그룹 선택자)로 각각 적용
div p : descendant(자손=자식+손주이하)다 적용
div + p : div옆에 가장 가까운 형제 하나에 css 적용
div ~ p : 일반형제 선택자 div옆에있는 모든형제에 css적용

Descendant Selector

The descendant selector matches all elements that are descendants of a specified element.
