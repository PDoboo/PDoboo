함수

함수는 영어로 Function 이라고 하는데 함수라고 했을때 떠오르는 이미지는 수납상자 입니다.
코드가 많아지면 그 코드들을 잘 정리 정돈하기 위한 도구가 필요합니다.
그것이 함수이며 유지보수와 작업시간을 단축시켜 주는 장점이 있습니다.

ON:이벤트

Event: 브라우저에서 사용자가 취하는 모든동작(마우스 클릭 등)
Event Handler - 이벤트가 발생했을때 자바스크립트 실행문을 실행하는것
Event Listener - 이벤트를 처리하는 함수
버튼을 누를때마다 배경이 바뀌고 다시 시작하기
키보드 접근성 - 마우스가 동작하지 않을때 키보드로 사용할 수 있어야 합니다.
